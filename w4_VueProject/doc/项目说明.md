# 项目说明

基于Vue框架的webApp项目 + 后台管理系统

## 项目类型
* 电商
* 金融
* 管理系统
* 资讯类App
* ...

## 项目要求
* 移动端适配
* 编写项目说明文档README.md
    * 项目名称
    * 演示
        * 官网
        * 上线网址
    * 团队与分工
        * 格式：组长：xxx，成员：xxx,xxx
        * 负责模块说明
    * 项目目录说明
* 编写数据API
* 爬取数据到本地数据库

## 技术栈

### 前端
* Vue
* Vue-Router
* Vuex
* Vue-Cli
* axios/fetch
* elementUI/MintUI/iView/antd
* git
* webpack

### 后端
* Nodejs
* Express
* MongoDB/mySQL
